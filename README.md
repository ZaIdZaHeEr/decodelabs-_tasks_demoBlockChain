# Demo Blockchain Project

A simple blockchain implementation built using Python for learning and educational purposes.  
This project demonstrates the core concepts behind blockchain technology such as blocks, hashing, proof of work, and chain validation.

---

# Features

- Block creation
- SHA-256 hashing
- Chain linking using previous hashes
- Proof of Work (PoW)
- Blockchain validation
- Tamper detection
- Simple transaction storage
- Educational implementation for beginners

---

# Technologies Used

- Python 3
- hashlib
- datetime
- json
